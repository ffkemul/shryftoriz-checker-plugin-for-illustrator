## <a name="українська"></a> 🇺🇦 Українська

Плагін для Adobe Illustrator, який автоматично знаходить, виділяє, закреслює або видаляє російські шрифти у ваших макетах. Працює на базі офіційного відкритого API проєкту [Shryftoriz](https://shryftoriz.online/).

### 💻 Сумісність
Плагін підтримує **Adobe Illustrator CC 2017 та всі новіші версії** (Windows та macOS).

### 📥 Встановлення

1. Завантажте файл **`ShryftorizChecker.zxp`** з розділу [Releases](https://github.com/ffkemul/shryftoriz-checker-plugin-for-illustrator/releases/latest).
2. Встановіть плагін за допомогою будь-якого ZXP-інсталятора, наприклад:
   * [ZXPInstaller](https://zxpinstaller.com/)
   * [Anastasiy's Extension Manager](https://install.anastasiy.com/)
3. Перезапустіть Adobe Illustrator.
4. Відкрийте плагін через меню: **Window > Extensions > Shryftoriz Checker** (Вікно > Розширення > Shryftoriz Checker).

### 🛠 Можливості

* **Жива база:** Плагін автоматично завантажує найсвіжішу базу шрифтів (1200+) безпосередньо з серверів Shryftoriz.
* **Гнучкі дії:** Знайдені шрифти можна:
  * Закреслити (Strikethrough)
  * Виділити (Select)
  * Розфарбувати (Recolor) у будь-який колір (з підтримкою HEX та HSB)
  * Повністю видалити з макета
* **Детальний звіт:** Після перевірки плагін видає точний список знайдених ворожих шрифтів, який можна скопіювати в один клік.
* **Двомовний інтерфейс:** Підтримка української та англійської мов.

### 🔗 Посилання
* Офіційний сайт бази: [shryftoriz.online](https://shryftoriz.online/)

## <a name="english"></a> EN English

An Adobe Illustrator plugin that automatically detects, selects, strikes through, or removes russian fonts in your designs. Powered by the official open API of the [Shryftoriz](https://shryftoriz.online/) project.

### 💻 Compatibility
The plugin supports **Adobe Illustrator CC 2017 and all newer versions** (Windows & macOS).

### 📥 Installation

1. Download the **`ShryftorizChecker.zxp`** file from the [Releases](https://github.com/ffkemul/shryftoriz-checker-plugin-for-illustrator/releases/latest) section.
2. Install the plugin using any ZXP installer, for example:
   * [ZXPInstaller](https://zxpinstaller.com/)
   * [Anastasiy's Extension Manager](https://install.anastasiy.com/)
3. Restart Adobe Illustrator.
4. Open the plugin via the menu: **Window > Extensions > Shryftoriz Checker**.

### 🛠 Features

* **Live Database:** Automatically fetches the latest font database (1200+) directly from Shryftoriz servers.
* **Flexible Actions:** Found fonts can be:
  * Struck through
  * Selected
  * Recolored (with HEX & HSB support)
  * Completely deleted from the document
* **Detailed Report:** Generates an exact list of found hostile fonts that can be copied in one click.
* **Bilingual Interface:** Supports both Ukrainian and English.

### 🔗 Links
* Official database website: [shryftoriz.online](https://shryftoriz.online/)
